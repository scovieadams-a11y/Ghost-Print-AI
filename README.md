## 📡 Overview
Ghost Print AI provides institutional-grade trading tools focused on **Liquidity Gaps**, **SMC**, and **Candle Range Theory**. The core objective is to eliminate emotional trading through high-concurrency algorithmic execution.

## 🛠 Features
* **PGA Indicator:** Real-time detection of institutional fair value gaps.
* **Execution Bot:** Automated risk management and Martingale-based recovery logic.
* **Premium Tech UI:** Clean, data-driven visuals for TradingView/MT5.

## 📥 Installation
1. Download the latest `.ex4`/`.ex5` or `.lua` file from the **Releases** tab.
2. Move the file to your terminal's `Indicators` or `Experts` folder.
3. Refresh your navigator and attach to the chart.
